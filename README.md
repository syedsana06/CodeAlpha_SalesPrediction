# CodeAlpha_SalesPrediction
Sales Prediction using Machine Learning 

## Conclusion

A Sales Prediction model was developed using Linear Regression to estimate product sales based on advertising expenditure across TV, Radio, and Newspaper platforms.

The dataset was cleaned and analyzed using exploratory data analysis techniques. Correlation analysis and visualization helped identify relationships between advertising channels and sales performance.

The model achieved a Mean Absolute Error (MAE) of 1.46, a Mean Squared Error (MSE) of 3.17, and an R² Score of 0.899. These results indicate that the model successfully captures the relationship between advertising investments and sales outcomes, explaining approximately 89.9% of the variation in sales.

This project demonstrates how machine learning can support marketing decisions by predicting sales performance based on advertising budgets.
